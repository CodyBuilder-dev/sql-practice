## repo의 구성
doit-oracle : Doit! 오라클 교재 폴더
1. data : 실습용으로 사용할 데이터베이스 및 메타데이터 DDL/DML
2. query : 실습용으로 사용할 SQL구문 파일
3. setting : 데이터베이스 내 유저 등 설정파일
